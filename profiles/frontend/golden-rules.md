# 前端专属黄金原则

> 补充全局 golden-principles.md，以下规则仅适用于前端代码。

**GP-FE-001** · 组件文件禁止直接调用 fetch/axios，必须通过 Hook 或 Service 层封装

**GP-FE-002** · 单个组件文件不超过 200 行，超出须拆分为子组件或提取 Hook

**GP-FE-003** · 禁止在组件内定义内联样式对象（每次渲染创建新引用），提取为常量或使用 CSS Module

**GP-FE-004** · 表单必须有 label 关联、图片必须有 alt、交互元素必须可键盘操作（a11y 基线）

**GP-FE-005** · 新增页面必须在 375px / 768px / 1440px 三个断点下验证布局
